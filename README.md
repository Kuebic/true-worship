# True Worship

Three short pieces on worship as the life of a true child — becoming one in heart with Heavenly Parent.

> **Worship is the life of a true child whose whole aim is to become one in heart with Heavenly Parent.**

- **Worship, in One Line** (`worship-crystal.md`) — the definition and its whole spine on one page.
- **How Worship Grew Up** (`worship-for-newcomers.md`) — for anyone: how worship matures from servant, to child, to true child.
- **Worship for Music Directors** (`worship-for-music-directors.md`) — for those who lead.

Served as a static site via GitHub Pages (built-in Jekyll, `jekyll-theme-cayman`).

Written from within Unificationist teaching. Shared freely.
